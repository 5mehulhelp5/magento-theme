Place demo images here. Anything under this folder is copied into pub/media/
preserving the relative path.

Example:
  media/wysiwyg/forever/furniture/promo-1.jpg  ->  pub/media/wysiwyg/forever/furniture/promo-1.jpg
  media/forever/bannerslider/furniture-slide-1.jpg -> pub/media/forever/bannerslider/furniture-slide-1.jpg

Also drop a "thumbnail.jpg" here (referenced by manifest.json) to show a preview
card in the admin Demo Importer grid.
